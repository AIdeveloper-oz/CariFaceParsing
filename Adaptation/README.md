# CariGan

We provide PyTorch implementations for caricature generation.

This code is heavily borrowed from (https://github.com/junyanz/pytorch-CycleGAN-and-pix2pix).


**Note**: The current software works well with PyTorch 0.4+.

## Prerequisites
- Linux or macOS
- Python 2 or 3
- CPU or NVIDIA GPU + CUDA CuDNN

## Getting Started
### Installation
- Install PyTorch 0.4+ and torchvision from http://pytorch.org and other dependencies (e.g., [visdom](https://github.com/facebookresearch/visdom) and [dominate](https://github.com/Knio/dominate)). You can install all the dependencies by
```bash
pip install -r requirements.txt
```
- Clone this repo:
```bash
git clone https://github.com/wenqingchu/CariGan
cd CariGan
```
